# EFA Tatto y Enric

Sitio frontend migrado a `Vite + React + React Router + Tailwind CSS v4`.

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

## Desarrollo

El servidor de desarrollo de Vite arranca por defecto en `http://localhost:5173`.

## Estructura

- `src/App.tsx`: rutas de la aplicación.
- `src/components/`: bloques visuales reutilizables.
- `src/styles/globals.css`: estilos globales y tema.
- `public/`: assets estáticos.
